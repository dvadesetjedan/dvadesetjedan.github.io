# Meetup starter kit

Mali Bitcoin susret je dovoljan početak.

## Minimalni format

1. Grad i okvirni datum.
2. Javna i jednostavna lokacija.
3. Kratka najava s Bitcoin-only pravilom.
4. RSVP ili Telegram koordinacija.
5. Miran ton za početnike.

Ne objavljuj privatne kontakte, seed phrase, iznose ili osjetljive lokacijske detalje.

## Dobra najava

Napiši što je tema, gdje i kada se ljudi nalaze, kome je susret namijenjen i gdje se prati zadnja promjena. Ne obećavaj investicijski rezultat.
